SISTEMAS OPERATIVOS Y REDES

CENTRO DE DATOS EN EL QUE SE INTERCONECTA 3 SUBREDES POR MEDIO DE UN ROUTER

En este proyecto se desean conectar 3 subredes a un mismo nodo central para que posteriormente se puedan conectar a internet, cada una de estas subredes cuenta con 10 ordenadores.

Para ejecutar y realizar este proyecto se hace uso de un router como nodo central que interconecta los 3 switches correspondientes a cada una de las subredes, para lograr que puedan enlazarse a la conexión Internet.
